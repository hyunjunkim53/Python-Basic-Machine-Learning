# 머신러닝 학습 기록

나도코딩 파이썬 활용편7(머신러닝) 강의 실습 내용 정리

## Scikit-Learn을 활용한 머신러닝 기초

### 구성

* **05 ~ 09**: 머신러닝 개요, 지도 학습, 선형 회귀
* **10 ~ 13**: 데이터 세트 분리, 경사 하강법
* **14 ~ 18**: 다중 선형 회귀, 원-핫 인코딩, 다중 공선성
* **19 ~ 24**: 회귀 모델 평가, 다항 회귀
* **25 ~ 28**: 로지스틱 회귀, 혼동 행렬
* **29 ~ 36**: 비지도 학습, K-평균(K-Means), 엘보우 방법, 유사도

## 프로젝트

### 구성

* **38 ~ 39**: 인구 통계학적 필터링 
* **40 ~ 45**: 콘텐츠 기반 필터링 (줄거리, 배우, 감독 등)
* **46 ~ 49**: 영화 추천 웹사이트
* **50 ~ 51**: 협업 필터링 (리뷰 기반)
